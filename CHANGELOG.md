## [1.0.0] - 2026-04-13

### Added
- Arquitectura modular con 30 módulos
- Base de migración hacia arquitectura state-driven
- Execution Verifier Engine integrado
- Capacidades IA base con LLaMA local
- Memoria semántica con embeddings
- Sistema de gamificación RPG
- Clasificación Core/Satélite
- Graceful shutdown y auto-restart
- Backpressure y circuit breaker
- Health checks integrados
- Perfiles de ejecución (minimal/standard/full)
- UI Telegram con menús interactivos
- Office Writer Automation con IA
- Blueprint Flow Inspector para análisis de conexiones

### Technical
- Runtime Node.js con message bus
- Módulos en Node.js y Python 3.11+
- Contrato de mensajes v2
- Separación de puertos semánticos
- Sistema de trazabilidad con trace_id
