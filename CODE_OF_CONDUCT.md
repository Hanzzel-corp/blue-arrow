# Código de Conducta

## Nuestro Compromiso

En el proyecto `blueprint-v0` (blue-arrow), nos comprometemos a mantener un ambiente abierto y acogedor para todos los contribuyentes.

## Estándares

### Comportamiento Esperado
- Ser respetuoso y considerado
- Aceptar críticas constructivas
- Enfocarse en lo que es mejor para la comunidad
- Mostrar empatía hacia otros miembros

### Comportamiento Inaceptable
- Uso de lenguaje o imágenes ofensivas
- Ataques personales o trolling
- Acoso público o privado
- Publicación de información privada sin consentimiento

## Responsabilidades

Los mantenedores del proyecto tienen el derecho y la responsabilidad de:
- Editar, eliminar o rechazar comentarios/commits
- Banear temporal o permanentemente a contribuyentes por comportamiento inapropiado

## Alcance

Este código aplica a:
- Issues y pull requests
- Discusiones en GitHub
- Comunicación directa relacionada al proyecto

## Aplicación

Casos de comportamiento inaceptable pueden reportarse a los mantenedores del proyecto.

## Atribución

Este Código de Conducta está basado en el [Contributor Covenant](https://www.contributor-covenant.org), versión 2.1.
